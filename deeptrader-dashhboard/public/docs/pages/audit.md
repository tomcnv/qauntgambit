# Audit Log

Complete audit trail of all configuration changes, user actions, and system events.

## Widgets & Cards

### Audit Event List
Chronological list of audit events with: timestamp, actor, action, target, severity, details.

### Severity Colors
Visual severity indicators: red (critical), amber (warning), green (info), gray (debug).

### Severity Icons
Icon badges per severity level for quick scanning.

## Actions

- Filter by severity, actor, action type, date range
- Search audit events by keyword
- Export audit log for compliance
- View event details
