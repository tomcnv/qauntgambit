# Bot Management

Create, configure, and manage trading bots across the fleet.

## Widgets & Cards

### Bot Cards
Grid of bot instances with status, strategy, exchange, PnL, and quick actions.

## Modals & Drawers

### Bot Startup Modal
Multi-step wizard for starting a bot: select exchange account, configure symbols, set risk params, confirm.

### Start Option Cards
Choose startup mode: Fresh Start, Resume Previous, Clone From.

### Account Policy Panel
Review and confirm account-level policies before bot activation.

## Actions

- Create a new bot instance
- Start/stop/pause a bot
- Clone a bot configuration
- Delete a bot instance
- View bot activity history
