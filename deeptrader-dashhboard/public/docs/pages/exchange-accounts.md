# Exchange Accounts

Manage exchange venue connections, API credentials, and account-level risk policies.

## Widgets & Cards

### Account Cards
One card per exchange account: exchange name, environment (live/testnet), balance, status, active bots count.

## Modals & Drawers

### Create Account Dialog
Form to add a new exchange account: select exchange, enter API credentials, set environment.

### Credentials Sheet
View and rotate API credentials for an account. Shows key status and last used timestamp.

### Policy Sheet
Configure account-level risk policies: max capital, allowed symbols, leverage limits.

## Actions

- Add a new exchange account
- Edit account credentials
- Enable/disable an account
- Set account risk policy
- View account balance history
