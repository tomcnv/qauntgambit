# Execution

Transaction Cost Analysis (TCA) dashboard for measuring and optimizing execution quality.

## Widgets & Cards

### TCA Metrics
Key metrics: Avg Slippage (bps), Avg Latency (ms), Fill Rate, Implementation Shortfall.

### Slippage Analysis
Distribution chart of slippage by symbol, time of day, and order size.

### Latency Breakdown
Waterfall chart: Signal → Decision → Order Submit → Exchange Ack → Fill.

## Actions

- Filter by symbol, exchange, time range
- Compare execution quality across exchanges
- Export TCA report
- Drill down into individual fills
