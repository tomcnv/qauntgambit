# Profiles

Manage strategy profiles that control trading behavior: signal parameters, risk settings, and execution preferences.

## Widgets & Cards

### Template Table
System-provided profile templates with strategy family, timeframe, and default parameters.

### User Profile Table
User-created and customized profiles with version tracking and activation status.

### Bulk Action Bar
Select multiple profiles for bulk operations: activate, deactivate, export, delete.

## Modals & Drawers

### Promotion Dialog
Promote a profile from draft to active with diff view showing changes from the base template.

## Actions

- Create a new profile from template
- Edit profile parameters
- Promote draft to active
- Bulk activate/deactivate
- Export/import profiles
- View profile version history
