# Settings

Central configuration hub for the entire platform, organized into specialized sub-tabs.

## Widgets & Cards

### Account Tab
User profile, display preferences, timezone, default currency.

### Billing Tab
Subscription plan, usage metrics, payment method, invoice history.

### Data Tab
Data feed configuration, retention policies, backfill settings, quality thresholds.

### Exchanges Tab
Exchange API key management, connection status, rate limit configuration.

### Notifications Tab
Alert channels (email, Slack, webhook), notification rules, severity thresholds.

### Risk Tab
Global risk parameters, default limits for new bots, circuit breaker defaults.

### Security Tab
Two-factor authentication, API key management, session management, audit preferences.

### System Health Tab
System diagnostics, service status, resource utilization, log level configuration.

### Trading Tab
Default trading parameters, order types, slippage tolerance, execution preferences.

## Actions

- Edit any setting value
- Save changes (with confirmation for critical settings)
- Reset to defaults
- Export/import configuration
- View change history
